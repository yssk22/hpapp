func Wrap ¶ 

func Wrap(ctx context.Context, inner error, options ...Option) error

Wrap returns a service error if the given inner error is not a service error. The message shown to the user would 'internal service error', but the message of the inner error is logged to the system. If the inner error is service error, it is returned directly.

type ServiceError ¶

ServiceError is an interface to avoid over wrapping by errors.Wrap If a service has a well defined error that can be passed to the clients/users, it should implement this interface so that client can handle the error properly. 

type ServiceError interface {
    DoNotWrap()
}
